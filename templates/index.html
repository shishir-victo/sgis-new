Student Attendance System API

API Endpoints

GET /api/students

Get a list of all students

POST /api/students

Add a new student with photo

{
    "name": "Student Name",
    "student_id": "S12345",
    "class_id": "C12345",
    "photo": "base64_encoded_image"
}

GET /api/classes

Get a list of all classes

POST /api/classes

Add a new class

{
    "name": "Class Name"
}

POST /api/take_attendance

Process classroom photo and mark attendance

{
    "class_id": "C12345",
    "date": "2023-07-01",
    "photo": "base64_encoded_image"
}

POST /api/manual_attendance

Manually mark attendance for a student

{
    "student_id": "S12345",
    "class_id": "C12345",
    "date": "2023-07-01",
    "status": true
}

GET /api/attendance_report

Get attendance report for a class on a specific date

Query parameters: class_id, date

GET /api/student_attendance_report

Get attendance report for a specific student

Query parameters: student_id
